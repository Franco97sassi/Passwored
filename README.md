## Description
Passwored is an app that locks access to other apps on your child's phone or tablet.

## Installation
1. Clone the repository:
   - git clone
2. Navigate to the project directory:
   - cd project
3. Install dependencies:
   - npm install

## Stack
- React
- Material UI
- Js
- Nodemailer
